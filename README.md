# AnimalMate 🐾

AnimalMate is a modern web platform that connects pet owners and breeders by helping them find compatible animals for mating based on breed, age, gender, health, and location.

## Features

- User Authentication
- Animal Registration
- Advanced Search
- Match Recommendation
- Image Uploads
- Messaging
- Location Based Search

## Tech Stack

Frontend
- React
- TypeScript
- TailwindCSS
- Framer Motion

Backend
- Spring Boot
- Spring Security
- PostgreSQL

Cloud
- Cloudinary
- Render
- Vercel
